# Flutter-Web-Code

## Deployment process

-`flutter clean`

-`flutter build web`

-`git add .`

-`git commit -m "Updates"`

-`git push origin main`

-`flutter pub global activate peanut`

-`flutter pub global run peanut:peanut`

-`git push --force origin --set-upstream gh-pages`
